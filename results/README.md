# Results

Validated outputs will be added after the refactored pipeline is run.

This repository will not repeat the original prototype's headline performance values as clinical evidence. Future result summaries will include:

- Cohort and class counts
- Missing-data summary
- Fold-specific metrics
- Mean and variability across folds
- Confusion matrices
- ROC and precision-recall curves
- Model and feature provenance
- Explicit limitations

Generated results and raw exports are excluded from version control unless they are privacy-safe, reproducible, and clearly documented.
